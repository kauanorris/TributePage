# 🎮 Overwatch Tribute Page

Uma página web criada como parte de um desafio do **FreeCodeCamp**, dedicada ao jogo Overwatch.  
O projeto apresenta informações sobre o jogo, seus heróis e modos de jogo, utilizando apenas **HTML e CSS**.

---

## 📌 Sobre o Projeto

Esta é uma **Tribute Page** inspirada no jogo Overwatch, desenvolvida com foco em prática de **estrutura HTML, organização de layout e estilização com CSS**.

A página contém:

- Navegação simples entre seções
- Informações sobre o jogo
- Cards com heróis do Overwatch
- Cards com modos de jogo
- Layout organizado utilizando Flexbox
- Imagens e tipografia personalizada

---

## 🚀 Tecnologias Utilizadas

- **HTML5**
- **CSS3**
- **Google Fonts**
- **Flexbox**

## 📷 Funcionalidades da Página

### 🏠 Home
Apresenta o título do jogo e uma pequena descrição.

### 📖 About
Seção explicando o que é o jogo Overwatch.

### 🦸 Heroes
Lista de heróis organizados em **cards**, contendo:

- Imagem
- Nome
- Classe
- Pequena descrição

### 🗺 Game Modes
Seção com os principais modos de jogo:

- Assault
- Escort
- Control
- Hybrid (Assault/Escort)

---

## 🧠 Objetivo do Projeto

Este projeto foi desenvolvido para praticar:

- Estruturação semântica em **HTML**
- Organização de **componentes visuais**
- Uso de **Flexbox**
- Criação de **cards responsivos**
- Organização de **assets (imagens e CSS)**
